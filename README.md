# upi-data-analysis
UPI Payments Data Analysis using Python, Pandas, Matplotlib and Seaborn


## Project Overview

This project analyzes UPI payment transaction data using Python and Pandas.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Findings

- UPI transactions showed strong growth trends.
- Digital payments increased significantly across categories.
- Peak transaction periods were identified.
- Transaction volume patterns were analyzed.
- Visual insights were generated using charts.

## Project Files

- Jupyter Notebook (.ipynb)
- Data Visualizations
- Analysis Report
